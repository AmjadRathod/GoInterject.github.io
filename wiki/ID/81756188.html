Function Summary

Used to specify a from value that will be passed to a target area in a new sheet. Used with PairGroup()() for sheets that are in the same workbook.

Function Arguments

Parameter NameDescriptionDefaultOptional

From

Enter a range or string that describes what will be copied from into the specified target

NO

TargetEnter a range or string related to a target cell that will recieve the values specified in the From parameter.
NO
RequireValueEnter a True or False to indicate if the From value is required to exist in the data. TRUEYES
ScopeRangeThis will limit the pair to only be used when the user is selected in the entered range.
YES

← From Range. This is range is context that will be brought into the target cell.

← Target sheet and cell

← The Required Value parameter is set to True, which requires the From value to exist.
